# Scout Report: tongkho-web Codebase

## Project Overview
- **Framework**: Astro 5.2 + React 19 (prerendered to static HTML)
- **Styling**: Tailwind CSS 3.4 + custom CSS
- **Language**: TypeScript 5.7 (strict mode)
- **Output**: Zero JavaScript (pure static HTML/CSS)
- **Build Time**: <5 minutes | Size: <3MB

## Project Structure
```
src/
├── components/         # Reusable UI components
│   ├── cards/         # property-card.astro (110 LOC)
│   ├── header/        # Navigation + mobile menu
│   ├── footer/        # Footer section
│   └── home/          # Homepage sections (hero, projects, properties, etc.)
├── layouts/           # base-layout.astro, main-layout.astro
├── pages/             # index.astro (homepage only)
├── types/             # Property, Project, NewsArticle, Location, SearchFilters
├── data/              # mock-properties.ts (255 LOC) - sample data
├── utils/             # format.ts - Vietnamese localization utilities
└── styles/            # global.css - Tailwind + custom styles
```

## Data Models
**Property Interface:**
- id, title, slug, type (apartment|house|villa|land|office|shophouse|warehouse)
- transactionType (sale|rent), price, priceUnit, area
- bedrooms?, bathrooms?, address, district, city
- images[], thumbnail, features[], createdAt, isFeatured, isHot

**Project Interface:**
- id, name, slug, developer, location, city, status (upcoming|selling|sold_out|completed)
- totalUnits, priceRange, area, amenities[], completionDate?, isFeatured

**NewsArticle Interface:**
- id, title, slug, excerpt, content, thumbnail
- category (market|policy|tips|project_news|investment), author, publishedAt, views

**Location Interface:**
- id, name, slug, image, propertyCount

## UI Components Available
1. **property-card.astro** (110 LOC) - Links to `/bat-dong-san/{slug}`, displays:
   - Image with hover zoom + gradient overlay
   - HOT/Featured badges (top-left)
   - Action buttons on hover (compare, share, favorite)
   - Price tag (bottom-left, gradient background)
   - Title + location + relative time
   - Features row: area (m²), bedrooms (PN), bathrooms (WC)

2. **hero-section.astro** - Hero banner with title/subtitle
3. **featured-project-section.astro** - Project cards grid (117 LOC)
4. **properties-section.astro** - Properties grid (64 LOC)
5. **locations-section.astro** - Location cards (120 LOC)
6. **news-section.astro** - News articles grid (101 LOC)
7. **hero-search.tsx** - React search form with tabs & filters (140 LOC)
8. **header.astro** - Navigation + mobile menu
9. **footer.astro** - Footer with links (173 LOC)

## Styling Approach
- **Colors**: Primary (orange #f97316), Secondary (slate #64748b)
- **Typography**: Inter (body), Be Vietnam Pro (headings)
- **Responsive**: <640px (mobile), 640-1024px (tablet), >1024px (desktop)
- **Tailwind Utilities**: Direct use (no component library)
- **Custom CSS**: `.card`, `.badge`, `.btn` classes in global.css

## Utilities & Formatting (src/utils/format.ts)
```typescript
formatPrice(price, unit) → "5.5 tỷ", "500 triệu", "15 triệu/tháng"
formatNumber(num) → Vietnamese thousand separators
formatDate(dateStr) → "28 tháng 1 năm 2026"
formatRelativeTime(dateStr) → "2 ngày trước", "Hôm qua"
truncateText(text, maxLength) → Ellipsis truncation
generateSlug(text) → Vietnamese to ASCII (dấu → no-dau)
```
